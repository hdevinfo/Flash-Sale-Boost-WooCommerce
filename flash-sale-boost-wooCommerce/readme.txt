## Flash Sale Boost for WooCommerce

Flash Sale Boost for WooCommerce -  Boost your e-commerce sales!

## Description

Use **Flash Sale Boost for WooCommerce** to boost your sale!

###Why use Flash Sale Boost for WooCommerce?
Easy to use, friendly, intuitive, organized, just install and you can start boost your sales.

###Flash Sale Boost** is a tool that allow you to show a countdown timer with your sale offer.

##Features
*  Countdown timer
*  Countdown timer for each product show on sale page
*  Custom Flash Sale Text

##Settings
*   Enter the value for font size, borders
*   Activate the Sale Text Status
*   Set the position of Countdown


##Installation

### From within WordPress
1. Visit 'Plugins > Add New'
1. Enter 'hdevinfo' in 'Search plugins...' form
1. Click 'Install Now' button and wait for install to finish
1. Click 'Activate' blue button 
1. Go to "FS Boost"  menu, left side, bottom

### Manually 
1. Upload the `plugin` folder to the `/wp-content/plugins/` directory
1. Activate the FS Boost plugin through the 'Plugins' menu in WordPress
1. Go to "FS Boost" menu.

### After activation ===
1. Go to FS Boost Menu
1. Configure fonts, position and activate (or not) the Sale Custom Text
1. Don't forget to set offer in woocommerce product schedule
